January 2019 - March 2019

Fabflix Movie Database - Web Application

Develop an e-commerce website similar to Amazon and eBay that lets users find their favorite movies, add movies to cart, and pay for movies. The movie search functionality uses MySQL Full-Text Search and has autocomplete features. The e-commerce website, Fabflix Movie Database - web application, is constructed using front-end and back-end separation architecture. The front-end is built from HTML, CSS, Bootstrap, Javascript, jQuery, and AJAX. The back-end is made up of a Tomcat web server that handles HTTP requests from client side and sends queries to the MySQL database management system. The Tomcat web server then receives movie data from the MySQL database and sends it to client side. The movie database - web application is deployed on a production server leveraging Amazon Web Services.
